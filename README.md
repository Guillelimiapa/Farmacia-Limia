# Farmacia Limia - Web

> **Proyecto ficticio** creado con fines educativos y demostrativos.

## 📌 Descripción

Este repositorio contiene una página web moderna, sencilla y funcional para **Farmacia Limia**, una farmacia ficticia ubicada en Ourense (Galicia).

El objetivo principal del proyecto es mostrar cómo crear una web profesional para una farmacia utilizando únicamente tecnologías web modernas y sin necesidad de backend.

## ✨ Características

- Diseño moderno y limpio con Tailwind CSS
- Logo personalizado con cruz verde de farmacia
- Horario de apertura destacado (9:00 - 21:00)
- Sección de **Consejos Farmacéuticos**
- Catálogo de servicios
- **Mapa interactivo** con Leaflet.js usando las coordenadas exactas de la farmacia
- Totalmente responsive (móvil, tablet y escritorio)
- Optimizada para GitHub Pages

## 🛠️ Tecnologías utilizadas

- **HTML5**
- **Tailwind CSS** (vía CDN)
- **Leaflet.js** (mapa interactivo)
- **Font Awesome** (iconos)
- JavaScript vanilla

## 🚀 Despliegue en GitHub Pages

Esta web está diseñada para publicarse fácilmente en **GitHub Pages**:

1. Sube el archivo `index.html` a la raíz del repositorio
2. Ve a **Settings → Pages**
3. Selecciona la rama `main` como fuente
4. ¡Tu web estará disponible en `https://tuusuario.github.io/nombre-del-repo`!
